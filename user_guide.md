## 界面布局

PasteLabel 界面主要分为以下几个部分：

1. **工具栏**（顶部）
   - 背景图加载按钮
   - 贴图加载按钮
   - 标签加载按钮
   - 绘制 BOX(W) 按钮
   - 显示 BOX(R) 复选框
   - 显示 Label(T) 复选框
   - 自动保存 (G) 复选框
   - 文件名前缀设置

2. **画布区域**（左侧）
   - 显示当前背景图片和贴图
   - 支持鼠标拖拽、缩放
   - 支持检测框绘制和编辑
   - 支持贴图位置调整

3. **控制面板**（右侧）
   - 背景图列表（带计数显示）
   - 标签列表（背景图标签/贴图标签）
   - 贴图列表（缩略图/列表视图）
   - 参数设置区域
   - 操作按钮区域

4. **状态栏**（底部）
   - 显示当前操作状态
   - 显示鼠标坐标

## 详细功能说明

### 1. 背景图片管理

#### 加载背景图片
**单张加载**:
1. 点击工具栏中的「背景图」图标按钮
2. 在弹出的文件选择对话框中选择一张或多张图片
3. 点击「打开」按钮

**批量加载（文件夹）**:
1. 点击工具栏中「背景图」图标右侧的文件夹图标
2. 选择包含图片的文件夹
3. 点击「选择文件夹」按钮

#### 切换背景图片
- 在背景图列表中点击图片
- 或使用快捷键：
  - `A` 键：切换到上一张背景图
  - `D` 键：切换到下一张背景图

**状态显示**:
- 背景图列表标题旁显示 `[当前序号 / 总数]`

---

### 2. 贴图功能

#### 加载贴图
**单张/批量加载**:
1. 点击工具栏中的「贴图」图标按钮
2. 选择一张或多张贴图
3. 点击「打开」按钮

**从文件夹加载**:
1. 点击「贴图」图标右侧的文件夹图标
2. 选择包含贴图的文件夹
3. 点击「选择文件夹」按钮

#### 添加贴图到画布
1. 在贴图列表中选择一个贴图
2. 贴图自动添加到背景图片中心
3. 鼠标拖动调整位置
4. 滚轮或右下角手柄调整大小

#### 随机贴图
1. 确保贴图列表中有贴图
2. 设置贴图参数：
   - **贴图个数**：1-20
   - **短边尺寸范围**：最小值 15-100，最大值 30-200
3. 点击「随机贴图」按钮
4. 系统自动随机放置贴图

#### 一键贴图（批量处理）
1. 确保已加载贴图
2. 点击「一键贴图」按钮
3. 系统从当前图片开始批量处理所有图片
4. 显示进度条，完成后自动保存

#### 清空画布
- 点击「清空画布」按钮清除当前图片的所有贴图

#### 贴图视图切换
- **缩略图视图**：直观预览贴图
- **列表视图**：显示贴图文件名
- 点击「列表视图」/「缩略图模式」按钮切换

---

### 3. 标签管理

#### 加载标签
1. 点击工具栏中的「标签」图标按钮
2. 选择标签文本文件（每行一个标签）
3. 点击「打开」按钮

#### 添加标签
1. 右键点击标签列表
2. 选择「增加标签」
3. 输入标签名称
4. 点击「确定」

#### 删除标签
1. 在标签列表中选择要删除的标签
2. 右键点击，选择「删除标签」
3. 点击「确定」
> **注意**：删除标签会同时从所有检测框中移除该标签

#### 修改标签
1. 右键点击标签列表
2. 选择「修改标签」
3. 输入新名称
4. 点击「确定」

#### 修改贴图标签
1. 选择画布上的贴图
2. 右键点击贴图
3. 在弹出菜单中选择标签

#### 自动添加标签
1. 勾选工具栏中的「贴图标签」复选框
2. 添加贴图时自动从文件名提取标签
3. 提取规则：按下划线 `_` 拆分，取第一部分
   - 示例：`标签_xxx.png` → 标签为"标签"

---

### 4. 检测框操作

#### 进入绘制模式
- 点击「绘制 BOX(W)」按钮
- 或按 `W` 键

#### 绘制检测框
1. 在背景图片上按住鼠标左键拖动
2. 释放鼠标完成绘制
3. 在弹出对话框中选择标签
4. 点击「确定」
> **注意**：检测框必须大于 3x3 像素

#### 调整检测框
**选中**: 点击检测框

**移动**: 拖动检测框

**调整大小**:
- 拖动四个角的手柄（绿色方块）
- 滚轮缩放（保持中心不变）

**修改标签**: 右键点击检测框，选择标签

#### 删除检测框
- 选中后按 `Delete` 或 `E` 键

#### 退出绘制模式
- 按 `Q` 键

#### 显示/隐藏检测框
- 点击「显示 BOX(R)」复选框
- 或按 `R` 键

#### 显示/隐藏标签名称
- 点击「显示 Label(T)」复选框
- 或按 `T` 键

---

### 5. 保存功能

#### 保存当前图片
1. 点击「保存图片」按钮
2. 保存到：
   - **目录**: `{原图目录}_paste_output/`
   - **文件名**: `{前缀}_{原文件名}`
3. 同时保存 JSON 标注文件

#### 保存所有图片
1. 点击「全部保存」按钮
2. 系统保存所有图片的修改版本
3. 显示进度条

#### 自动保存
1. 勾选「自动保存 (G)」复选框
2. 切换图片时自动保存有贴图的图片
3. 按 `G` 键快速切换自动保存

#### 保存文件名设置
1. 勾选「添加文件名前缀」复选框
2. 在输入框中输入前缀（默认 "paste"）
3. 保存的文件名包含前缀

#### 输出文件格式

**保存的图片**:
- **位置**: `{原图目录}_paste_output/{前缀}_{原文件名}`
- **内容**: 背景图 + 所有贴图

**JSON 标注文件**:
- **位置**: `{原图目录}_paste_output/{前缀}_{原文件名}.json`
- **格式**: Labelme 格式
- **内容**:
```json
{
    "version": "5.0.1",
    "flags": {},
    "shapes": [
        {
            "label": "标签名",
            "points": [[x1,y1], [x2,y1], [x2,y2], [x1,y2]],
            "group_id": null,
            "description": "",
            "shape_type": "rectangle",
            "flags": {}
        }
    ],
    "imagePath": "文件名",
    "imageData": null,
    "imageHeight": 高度，
    "imageWidth": 宽度
}
```
- **包含**: 所有贴图的位置和标签 + 所有检测框的位置和标签

### 6. 参数设置

#### 贴图参数
在右侧控制面板的「贴图参数设置」区域：

**贴图个数**:
- 范围：1-20
- 默认值：1

**短边尺寸范围**:
- **最小值**：15-100 像素
- **最大值**：30-200 像素
- **自动限制**：系统自动确保最小值 ≤ 最大值

#### 显示选项
- **显示 BOX(R)**: 显示/隐藏检测框（快捷键：`R`）
- **显示 Label(T)**: 显示/隐藏标签名称（快捷键：`T`）
- **贴图标签**: 启用/禁用自动标签功能

#### 保存选项
- **添加文件名前缀**: 是否在文件名前添加前缀
- **前缀输入**: 自定义前缀（默认 "paste"）

---

### 7. 其他功能

#### 背景图缩放
- **滚轮缩放**: 滚动鼠标滚轮缩放背景图
- **重置缩放**: 按住 `Ctrl` 键并滚动鼠标滚轮
- **缩放范围**: 0.5x - 3.0x

#### 状态显示
- **状态栏**: 显示当前操作状态
- **鼠标坐标**: 显示画布上的鼠标位置
- **参数提示**: 参数调整越界时显示提示

#### 文件计数显示
- 背景图列表标题旁显示 `[当前序号 / 总数]`

## 🔧 配置说明

所有配置文件位于 `PasteY/config.py`

### 主要配置项

```python
# 支持的图片格式
SUPPORTED_IMAGE_EXTENSIONS = ['.png', '.jpg', '.jpeg', '.bmp', '.gif']

# 缩略图配置
THUMBNAIL_CONFIG = {
    'grid_width': 56,           # 网格宽度
    'grid_height': 60,          # 网格高度
    'spacing': 2,               # 间距
    'thumb_size': (48, 48),     # 缩略图尺寸
}

# 贴图参数
PASTE_PARAMS = {
    'default_count': 1,         # 默认数量
    'min_count': 1,             # 最小值
    'max_count': 20,            # 最大值
    'default_min_size': 30,     # 默认最小尺寸
    'default_max_size': 60,     # 默认最大尺寸
    'min_size_range': (15, 100),  # 最小值范围
    'max_size_range': (30, 200),  # 最大值范围
}

# 背景图缩放
BACKGROUND_SCALE_CONFIG = {
    'default_scale': 1.0,       # 默认缩放
    'min_scale': 0.5,           # 最小缩放
    'max_scale': 3.0,           # 最大缩放
    'step': 0.1,                # 步进值
}

# 检测框配置
DETECTION_BOX_CONFIG = {
    'handle_size': 8,           # 手柄大小
    'border_width': 2,          # 边框宽度
    'colors': [                 # 边框颜色列表
        '#FF0000', '#00FF00', '#0000FF', 
        '#FFFF00', '#00FFFF', '#FF00FF'
    ],
}

# 贴图配置
PASTE_CONFIG = {
    'border_color': '#00FF00',  # 边框颜色
    'min_size': 10,             # 最小尺寸
    'opacity': 200,             # 透明度
}

# 窗口配置
WINDOW_CONFIG = {
    'default_width': 1600,      # 默认宽度
    'default_height': 1000,     # 默认高度
    'min_width': 1024,          # 最小宽度
    'min_height': 768,          # 最小高度
    'panel_width': 300,         # 右侧面板宽度
}

# 自动保存配置
AUTO_SAVE_CONFIG = {
    'enabled': False,           # 默认是否启用
    'delay': 500,               # 延迟（毫秒）
}
```

### 修改配置

1. 打开 `PasteY/config.py` 文件
2. 修改相应的配置值
3. 保存文件
4. 重新运行程序即可生效

**注意**: 建议只通过配置文件调整参数，不要硬编码在代码中。

## 🐛 常见问题

### 1. 无法启动程序

**错误信息**: `ModuleNotFoundError: No module named 'PyQt5'`

**解决方法**:
```bash
pip install PyQt5 opencv-python
```

### 2. 贴图不显示

**可能原因**:
- 未加载贴图
- 贴图不在当前视图范围内
- 背景图缩放比例过小

**检查步骤**:
1. 确认已加载贴图
2. 滚动鼠标滚轮放大视图
3. 拖动画布查找贴图
4. 检查贴图是否被其他元素遮挡

### 3. 检测框无法绘制

**可能原因**:
- 未进入绘制模式
- 未在背景图范围内点击
- 检测框尺寸过小

**检查步骤**:
1. 确认已点击「绘制 BOX」或按 `W` 键
2. 确保在背景图范围内按住鼠标拖动
3. 拖动距离应大于 3x3 像素
4. 检查是否成功弹出标签选择对话框

### 4. 保存失败

**可能原因**:
- 输出目录无写入权限
- 磁盘空间不足
- 文件名包含非法字符

**解决方法**:
1. 检查文件夹写权限
2. 清理磁盘空间
3. 避免使用特殊字符命名
4. 确认原图文件存在

### 5. 尺寸参数调整出现越界

**现象**: 调整贴图尺寸参数时提示越界

**解决方法**:
- 系统会自动限制范围，无需手动处理
- 确保最小值范围：15-100
- 确保最大值范围：30-200
- 系统会自动确保最小值不大于最大值

### 6. 标签无法删除

**可能原因**:
- 标签正在被使用

**解决方法**:
- 删除标签会同时删除所有使用该标签的检测框
- 这是正常行为，防止出现无主检测框

### 7. 一键贴图无反应

**可能原因**:
- 未加载贴图
- 未加载背景图

**检查步骤**:
1. 确认已加载背景图和贴图
2. 确认当前不是第一张图片（从当前图片开始处理）
3. 观察进度条是否显示
