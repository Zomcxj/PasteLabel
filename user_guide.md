## 界面布局

PasteLabel 界面分为四个区域：

1. **工具栏**（顶部）— 背景图/贴图/标签加载按钮、选项菜单、统计按钮、标注/贴图模式切换、语言/主题/设置按钮
2. **画布区域**（左侧）— 显示背景图、贴图、检测框，支持缩放/平移/绘制
3. **控制面板**（右侧）— 背景图列表（步长翻页 + 工作/移除路径切换）、标签列表、贴图列表、操作按钮
4. **状态栏**（底部）— 实时显示鼠标坐标/贴图数/检测框数/标签统计

---

## 1. 贴图/标注模式切换

工具栏右侧的双栏滑动切换控件，控制画布交互的优先级。**软件默认打开为标注模式。**

| | 标注模式（默认） | 贴图模式 |
|---|---|---|
| **悬停高亮** | 鼠标移到检测框上自动选中 | 鼠标移到贴图上自动选中 |
| **左键优先级** | 检测框 → 贴图项 → 背景 | 贴图项 → 检测框 → 背景 |
| **画布内放大预览** | 开启后，选中检测框或绘制时，鼠标位于背景图内自动显示 | 不显示 |

切换时状态栏显示 `Mode: Annotate` / `Mode: Paste`（2秒后消失）。

## 2. 背景图片管理

### 加载背景图片
- **单张加载**：点击「背景图」图标按钮，选择图片
- **批量加载**：点击文件夹图标，选择图片文件夹（自然排序）

### 切换背景图片
- 点击背景图列表中的图片
- 或使用快捷键：`D` 下一张、`A` 上一张
- **步长设置**：数字滚轮 1~10，每次翻页的图片数量

### 工作路径 / 移除路径
- 点击「工作路径」/「移除路径」按钮切换视图
- 工作路径：显示参与标注的图片，所有操作可用
- 移除路径：显示被移除的图片，右键仅可恢复，W/Q/Delete 快捷键禁用
- 移除路径为空时可安全切换，不会报错

### 拖拽操作
- **拖入**：将图片（PNG/JPG/BMP/GIF）或 LabelMe JSON 拖入窗口，自动加载
- **拖出**：从画布按住左键拖到窗口外，复制当前背景图到桌面/文件夹

## 3. 记忆

点击工具栏「记忆」按钮打开最近素材记录窗口。

### 自动记录
- 关闭软件时自动保存当前背景图路径 + 贴图路径 + 标签文件路径 + 图片索引
- 最多 10 组记录，按路径组合去重（同一组再次使用时更新到顶部）

### 操作
- **加载** — 清空当前素材后加载该组记录，自动跳到记录的图片序号
- **修改备注** — 按任务/数据集/客户名填写备注
- **删除** — 删除选中记录

## 4. 贴图功能

### 加载贴图
- **单张/批量**：点击「贴图」图标按钮，选择贴图文件
- **文件夹加载**：点击文件夹图标，选择贴图文件夹

### 手动贴图
1. 在贴图列表中点击贴图 → 自动放置到背景中心
2. 鼠标拖动调整位置
3. 滚轮缩放 或 拖拽右下角手柄调整大小

### 随机贴图
1. 设置参数：**贴图个数**（1~20）、**短边尺寸范围**（最小 15~100，最大 30~200）
2. 点击「随机贴图」— 中心点避让算法自动放置，避免与检测框/已贴图重叠

### 一键贴图（批量）
点击「一键贴图」，从当前图片开始逐张执行随机贴图，弹出进度条。

### 贴图列表视图
- 缩略图视图（预览缩略图） ↔ 列表视图（显示文件名）

---

## 5. 标签管理

### 加载标签
点击「标签」图标按钮，选择标签文本文件（每行一个标签）。

### 标签列表操作（右键）
两个标签列表都支持右键菜单：

| 操作 | 背景图标签 | 贴图标签 |
|------|-----------|---------|
| 增加标签 | 输入新名称，加入全局标签 | 输入新名称，加入贴图标签列表 |
| 修改标签 | 重命名，级联更新所有检测框 JSON | 重命名，级联更新所有使用该标签的贴图项 |
| 删除标签 | 删除，级联移除所有背景图中使用该标签的检测框 | 删除，级联移除使用该标签的贴图项 |

> 标签列表按数量降序排列，统计仅计算工作路径的图片。

### 自动添加标签
勾选「贴图标签」复选框后，添加贴图时自动从文件名提取标签（按 `_` 拆分取第一部分）。

## 6. 检测框操作

### 画布内放大预览
开启「窗口放大器」后，仅在标注模式下选中检测框或进入绘制模式、且鼠标位于背景图内时，画布内显示鼠标附近的放大预览，辅助精准绘制与调整。放大倍率可在设置对话框中调节（0.8× ~ 3.0×）。

### 进入绘制模式
点击「绘制 BOX(W)」或按 `W` 键。

### 绘制检测框
1. 在背景图上按住鼠标左键拖动
2. 释放鼠标完成绘制
3. 弹出标签选择对话框，选择/输入标签

> 检测框最小 3×3 像素。

### 调整检测框
- **选中**：点击检测框
- **移动**：拖动检测框
- **缩放**：拖动四角手柄（绿色方块）；选中后在框内滚动滚轮可等比缩放，框外滚动则调整距离鼠标最近的一条边
- **滚轮参数**：框内缩放步长和框外单侧位移像素均可在设置对话框中调整

### 标注十字线
- 仅在标注模式、鼠标位于画布内且背景图已显示时出现
- 可在设置对话框调整粗细（0.5~3.0）、透明度（0~255）和颜色

### 显示/隐藏
- 显示/隐藏检测框边框：`R` 键 或 选项菜单
- 显示/隐藏标签名称：`T` 键 或 选项菜单

## 7. 右键菜单

### 画布右键

| 点击位置 | 菜单内容 |
|---------|---------|
| 贴图项 | 移除该贴图（或按 Delete）+ 分隔线 + 贴图标签列表（快速切换标签）+ 增加标签 |
| 检测框 | 移除（带撤销）+ 修改标签 + 分隔线 + 检测框标签列表（快速切换标签） |
| 背景空白（工作视图） | 移除图片（移到 `_delete_` 文件夹） |
| 背景空白（移除视图） | 恢复到工作路径 |

> 隐藏标签状态下（`R` 键关闭），右键检测框不弹修改菜单，移除图片优先。

## 8. 保存功能

### 保存当前图片
点击「保存图片」或 `Ctrl+S`，保存到 `{原图目录}_paste_output/`。

### 保存所有图片
点击「全部保存」或 `Ctrl+Shift+S`，弹出进度条逐张保存。

### 输出格式
- 合成图：背景 + 贴图绘制到一张图上
- JSON：LabelMe v5.0.1 格式，shapes 包含贴图矩形 + 检测框
- 文件名：`{前缀}_{原文件名}.{ext}`

### 自动保存
勾选「自动保存(G)」后，切换图片时静默保存当前编辑。

## 9. 选项菜单

点击工具栏「选项」按钮打开下拉菜单：

| 菜单项 | 快捷键 | 说明 |
|-------|--------|------|
| 绘制 BOX | W | 进入/退出检测框绘制模式 |
| 显示检测框 | R | 显示/隐藏检测框边框 |
| 显示 Label | T | 显示/隐藏标签名称 |
| 显示网格 | Ctrl+G | 显示/隐藏网格辅助线 |
| 显示贴图名 | F | 显示/隐藏贴图文件名 |
| 自动保存 | G | 切换自动保存状态 |
| 添加文件名前缀 | — | 保存时文件名是否添加前缀 |
| 画布图片复制 | — | 开启后，从画布按住左键拖到窗口外可复制当前背景图到桌面/文件夹 |
| 窗口放大器 | — | 开启后，标注模式下选中检测框或绘制时，鼠标位于背景图内显示画布内放大预览 |

## 10. 网格辅助线

- 按 `Ctrl+G` 显示/隐藏网格
- 网格为虚线点阵，默认间距 50px
- 可在设置对话框中调整线宽（1~10）和透明度（0~255）
- 缩放间距 < 5px 时自动隐藏
- 深色/浅色主题使用不同颜色值

---

## 11. 撤销重做

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销上一步操作 |
| Ctrl+Y | 重做 |

支持撤销的操作：添加贴图、清空画布、随机贴图、绘制检测框、拖拽/缩放检测框和贴图。最多保存 50 条历史。

---

## 12. 标签统计

点击工具栏「统计」按钮打开标签统计弹窗，包含两个表格：

- **背景图标签** — 显示类别名称和数量，按数量降序
- **贴图标签** — 显示贴图类别和数量
- **颜色替换** — 点击颜色按钮可替换类别颜色，按钮显示当前 HTML色值并立即更新

表格左上角角按钮颜色跟随深色/浅色主题。

---

## 13. 设置对话框

点击工具栏 ⚙ 按钮打开。

### 快捷键设置
- 点击输入框 → 按新快捷键组合 → 保存
- 支持重置单个快捷键到默认值
- 可自定义设置中列出的动作和缓存槽快捷键
- 打开设置时自动禁用主窗口快捷键，防止 Ctrl+F 等冲突

### 其他设置
- 文件名前缀（默认 "paste"）
- 网格线粗细和透明度
- 状态栏显示的标签类别数
- 窗口放大器倍率、检测框滚轮编辑参数
- 检测框缩放步长只控制检测框滚轮缩放；贴图缩放步长只控制贴图滚轮缩放
- 标注十字线粗细、透明度和颜色

## 快捷键大全

| 快捷键 | 功能 |
|--------|------|
| A | 上一张背景图 |
| D | 下一张背景图 |
| W | 进入/退出绘制检测框模式 |
| Q | 取消绘制 |
| R | 显示/隐藏检测框 |
| T | 显示/隐藏标签名 |
| F | 显示/隐藏贴图名 |
| G | 切换自动保存 |
| Delete | 删除选中的检测框/贴图 |
| ← ↑ → ↓ | 移动选中的检测框或贴图（步长在参数设置中调整） |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+S | 保存当前图片 |
| Ctrl+Shift+S | 保存所有图片 |
| Ctrl+G | 显示/隐藏网格 |
| Ctrl+F | 适应视图 |
| Ctrl++ | 放大 |
| Ctrl+- | 缩小 |
| Ctrl+Shift+D | 移除当前图片到 `_delete_` |
| Ctrl+Shift+R | 从 `_delete_` 恢复图片 |
| Ctrl+C | 复制选中的检测框到可写入缓存槽 |
| Ctrl+V | 粘贴当前缓存槽中的检测框 |
| 1 / 2 / 3 | 粘贴对应缓存槽中的检测框（默认） |

> 设置中列出的动作和缓存槽快捷键可自定义修改。缓存槽支持持久化、锁定和改名；移除路径视图下 W/Q/Delete 被禁用。

## 常见问题

### 启动报错 ModuleNotFoundError
```bash
pip install PyQt5 opencv-python
```

### 启动 exe 缺少 ffi.dll
Windows 深色标题栏依赖 `ffi.dll`。使用 `pastelabel/build.sh` 打包会自动包含，或手动复制 `<conda_env>/Library/bin/ffi.dll` 到 exe 同目录。

### 贴图不显示
确认已加载贴图，滚轮放大视图，检查贴图是否在画布范围外。

### 检测框无法绘制
确认已按 `W` 进入绘制模式，在背景图范围内拖动，框需大于 3×3 像素。

### 保存失败
检查输出目录写权限、磁盘空间、文件名合法性。

### 一键贴图无反应
确认已加载背景图和贴图，从当前图片开始处理。

### 标签删除后消失
删除标签会级联移除所有使用该标签的检测框，这是设计行为。保存后 JSON 同步更新。

### 发生未处理异常或 Qt 报错
未处理异常和 Qt 消息会写入用户主目录的 `~/pastelabel.log`，可将该日志用于排查问题。
