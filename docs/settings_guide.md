# 设置说明

[返回用户指南](./user_guide.md)

设置对话框分为两个页面：**快捷键设置** 和 **参数设置**。

---

## 快捷键设置

在「设置」对话框的「快捷键设置」页面，可自定义以下 20 个快捷键。点击输入框后按键即可绑定，支持 `Ctrl+`、`Alt+`、`Shift+` 组合键。

| 操作 | 默认值 | 说明 |
|------|--------|------|
| 撤销 | `Ctrl+Z` | 撤销上一步操作 |
| 重做 | `Ctrl+Y` | 重做 |
| 显示网格线 | `Ctrl+G` | 显示/隐藏网格辅助线 |
| 显示BOX | `R` | 显示/隐藏检测框边框 |
| 显示标签名 | `T` | 显示/隐藏标签名称 |
| 显示贴图名 | `F` | 显示/隐藏贴图文件名 |
| 自动保存B | `G` | 切换自动保存合成图 |
| 自动保存P | `H` | 切换自动保存 JSON |
| 绘制检测框 | `W` | 进入/退出检测框绘制模式 |
| 退出绘制 | `Q` | 取消绘制 |
| 下一张 | `D` | 下一张背景图 |
| 上一张 | `A` | 上一张背景图 |
| 删除选中 | `E` | 删除选中的检测框/贴图 |
| 适应视图 | `Ctrl+F` | 适应视图 |
| 放大 | `Ctrl++` | 放大画布 |
| 缩小 | `Ctrl+-` | 缩小画布 |
| 移除图片 | `Ctrl+Shift+D` | 移除当前图片到 `_delete_` |
| 恢复图片 | `Ctrl+Shift+R` | 从 `_delete_` 恢复图片 |
| 缓存槽1 | `1` | 粘贴缓存槽1中的检测框 |
| 缓存槽2 | `2` | 粘贴缓存槽2中的检测框 |
| 缓存槽3 | `3` | 粘贴缓存槽3中的检测框 |

> 缓存槽快捷键可自定义修改；移除路径视图下 `W`/`Q`/`Delete` 被禁用。

## 参数设置

在「设置」对话框的「参数设置」页面，可调整以下参数。

### 文件名前缀

- **添加文件名前缀** — 保存时文件名是否添加前缀，前缀值可在此修改（默认 `paste`）

### 窗口放大器

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 窗口放大器倍率 | 0.8 ~ 3.0 | 2.0 | 放大预览区域的缩放倍率 |
| 窗口放大器尺寸 | 80 ~ 400 px | 160 | 放大预览窗口的尺寸 |
| 放大器显示位置 | 侧边 / 鼠标中心 | 侧边 | 侧边：放大窗口在鼠标右下方偏移显示；鼠标中心：以鼠标为中心显示，始终可见 |

### 网格辅助线

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 网格线粗细 | 1 ~ 10 | 1 | 网格线宽度（像素） |
| 网格线灰度 | 0 ~ 255 | 120 | 网格线透明度，0 为完全透明，255 为完全不透明 |

### 状态栏

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 状态栏类别数 | 1 ~ 20 | 3 | 状态栏最多显示的标签类别数量 |

### 检测框

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 类别名字号 | 5 ~ 15 | 9 | 检测框标签名的字体大小 |
| 标签名位置 | 框外侧 / 框内侧 | 框外侧 | 标签名显示在检测框外侧或内侧 |
| 缩放句柄大小 | 7 ~ 15 | 8 | 检测框四角手柄的尺寸（同时影响贴图句柄） |
| 框线粗细 | 1 ~ 4 | 1.0 | 检测框和贴图的边框宽度 |

### 移动与缩放

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 移动步长 | 1 ~ 5 | 1 | 方向键（←↑→↓）移动选中的检测框或贴图的步长（像素） |
| 检测框缩放步长 | 0.01 ~ 0.30 | 0.05 | 在检测框内滚动滚轮时的整体缩放比例步长 |
| 贴图缩放步长 | 0.01 ~ 0.30 | 0.15 | 在贴图内滚动滚轮时的整体缩放比例步长 |
| 单侧位移像素 | 1 ~ 50 | 5 | 在检测框外滚动滚轮时，调整距离鼠标最近一条边的位移像素 |

> 检测框缩放步长只控制检测框滚轮缩放，贴图缩放步长只控制贴图滚轮缩放，两者独立。

### 标注十字线

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 十字线粗细 | 0.5 ~ 3.0 | 1.0 | 标注模式下十字线的宽度 |
| 十字线透明度 | 0 ~ 255 | 160 | 十字线的透明度 |
| 十字线颜色 | 色值选择 | `#00FF80` | 点击色块按钮选择颜色 |

十字线仅在标注模式、鼠标位于画布内且背景图已显示时出现。
